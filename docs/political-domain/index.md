# 政治ドメイン知識

このセクションでは、Sagebaseの開発に必要な日本の政治制度に関するドメイン知識をまとめています。

## 概要

日本の議会制度は、国会と地方議会で大きく異なる運営サイクルを持っています。Sagebaseでデータをモデリングする際は、これらの違いを理解しておくことが重要です。

## コンテンツ

- [国会の仕組み](national-diet.md) - 国会の会期制度について
- [地方議会の仕組み](local-assembly.md) - 地方議会と統一地方選挙について
- [首長選挙](executive-elections.md) - 知事・市長選挙のサイクルについて

## Sagebaseでのデータモデリングへの影響

| 議会種別 | メンバー更新サイクル | データモデル上の考慮点 |
|----------|---------------------|----------------------|
| 国会 | 会期ごと（年単位） | `term`に「第○○回国会」を設定 |
| 地方議会 | 4年ごと（統一地方選挙） | `Election`で選挙を管理し、`term`に「第○期」を設定 |
| 首長 | 個別のサイクル | 議会とは別に管理が必要 |
