# 総則の設定内容とその理由

### 第1条（名称）
この法人は、**一般社団法人政治ベース**と称する。

### 第2条（主たる事務所）
この法人は、主たる事務所を京都府京都市に置く。

### 第3条（目的）
この法人は、営利を目的とせず、民主主義社会における政治情報の透明性向上と市民の政治参加の促進に寄与することを目的とし、日本の国会および地方議会における政治議論データを収集・整備し、中立かつ公平な立場で構造化されたオープンデータとして広く社会に無償で提供することを目的とする。

### 第4条（事業）
この法人は、前条の目的を達成するため、次の事業を行う。

1. 国会、地方議会及び行政機関に関する情報の収集、構造化、蓄積及び提供
2. 政治家、候補者及び政党等の情報のデータベース化及び公開
3. 政治・行政データの分析、可視化のためのソフトウェア及びプラットフォームの開発、運営
4. 政治データの標準化及びオープンデータ化の推進
5. 前各号に掲げる事業に附帯又は関連する事業

!!! note "補足説明"
    **「前各号に掲げる事業に附帯又は関連する事業」という一文は、法律上の「お守り」**として必ず入れておくべきです。

    これがないと、例えば以下のような「些細だけど必要な事務」が、厳密には「定款違反」と突っ込まれるリスクが出てきます。

    - プロダクトを維持するためのサーバー代の管理
    - 事務手数料のやり取り
    - データのライセンスに関する契約締結

    「メインの1〜4をやるために必要な、細々とした事務も含まれますよ」という、事務的なバッファなので、実務上は削らないのが鉄則です。

### 第5条（公告方法）
1. この法人の公告は、電子公告により行う。
2. 事故その他やむを得ない事由によって電子公告による公告をすることができないときは、官報に掲載する方法により行う。

!!! note "補足説明"
    **なぜ「官報」を予備に入れるのか？**

    ウェブサイトのサーバーが落ちている期間に、法律で決められた公告（決算公告など）の期限が来てしまうと「公告義務違反」になるリスクがあります。そのため、万が一の予備として「官報（国が発行する新聞）」をサブに指定しておくのが、登記実務上の「王道」です。

    ---

    **ウェブサイトで行う際の「実務的な注意点」**

    ここが非常に重要です。ウェブ公告を選ぶ場合、以下の手間とコストが発生します。

    **① URLの登記が必要**

    「電子公告にします」と定款に書く場合、法務局での登記の際に、公告を掲載するページのURL（例：`https://example.org/public-notice`）を届け出る必要があります。

    **② 登記費用（登録免許税）が変わる**

    通常の設立登記費用に加え、URLを登記するための登録免許税として3万円が追加でかかります。

    もし将来、ドメインを変えたりURLを変更したりすると、その都度3万円の登記費用を払ってURL変更の手続きをしなければなりません。

    **③ 5年間消せない（決算公告の場合）**

    一般社団法人は、毎年度「貸借対照表（決算公告）」を公開する義務があります。ウェブサイトに掲載する場合、5年間は継続して誰でも見られる状態にしておかなければならないというルールがあります。
