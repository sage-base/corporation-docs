# 定款の変更及び解散の設定内容とその理由

### 第37条（定款の変更）
この定款は、社員総会の決議によって変更することができる。

### 第38条（解散）
この法人は、社員総会の決議その他法令で定められた事由により解散する。

### 第39条（残余財産の帰属）
この法人が清算をする場合において有する残余財産は、社員総会の決議を経て、公益社団法人及び公益財団法人の認定等に関する法律第5条第17号に掲げる法人又は国若しくは地方公共団体に贈与するものとする。

!!! info "残余財産の帰属先について"
    第39条で残余財産の帰属先を「公益社団法人・公益財団法人、国、地方公共団体」に限定しているのは、非営利型法人の要件を満たすためです。

    解散時に残った財産を社員や特定の個人に分配できる規定にしてしまうと、実質的に「利益の分配」が可能な法人とみなされ、非営利型法人としての税制上の優遇措置を受けられなくなります。
