# 計算の設定内容とその理由

### 第30条（事業年度）
この法人の事業年度は、毎年2月1日から翌年1月31日までの年1期とする。

!!! info "事業年度を2月始まりとした理由"
    当法人の設立予定日は2月中です。もし事業年度を一般的な4月始まり（3月決算）とした場合、設立初年度がわずか2か月弱で終了してしまい、売上の有無にかかわらず設立直後の5月末までに初回の法人税申告が必要となります。法人設立直後の忙しい時期に申告期限が来ることを避けるため、設立月である2月を事業年度の開始月とし、初年度を丸々1年間（2月1日から翌年1月31日まで）確保できるようにしています。

### 第31条（事業計画及び収支予算）
1. この法人の事業計画書、収支予算書については、毎事業年度の開始の日の前日までに、代表理事が作成し、社員総会の承認を受けなければならない。これを変更する場合も同様とする。
2. 前項の規定にかかわらず、やむを得ない理由により予算が成立しないときは、代表理事は、予算成立の日まで前事業年度の予算に準じ収入支出することができる。
3. 前項の収入支出は、新たに成立した予算の収入支出とみなす。

!!! info "予算の事前承認について"
    本条では、事業年度が始まる前に社員総会で予算の承認を得るルールを定めています。毎年1月末までに必ず総会を開く必要があり事務負担は増しますが、ガバナンスが効き透明性が高まるため、当法人の「政治の透明性」という理念（第3条）に合致した運用となります。また、第2項・第3項で予算が成立しない場合の暫定措置を定めており、実務上の柔軟性も確保しています。

### 第32条（事業報告及び決算）
1. この法人の事業報告及び決算については、毎事業年度終了後、代表理事が次の書類を作成し、定時社員総会に提出し、第1号及び第2号の書類についてはその内容を報告し、第3号から第5号までの書類については承認を受けなければならない。
    1. 事業報告
    2. 事業報告の附属明細書
    3. 貸借対照表
    4. 正味財産増減計算書
    5. 貸借対照表及び正味財産増減計算書の附属明細書
2. 前項の書類のほか、次の書類を主たる事務所に10年間備え置くものとする。
    1. 定款
    2. 社員名簿

!!! info "書類名称について"
    株式会社では「損益計算書（P/L）」を作成しますが、一般社団法人の会計基準では、これに相当する書類を「正味財産増減計算書」と呼びます。正式な会計基準に合わせ、「正味財産増減計算書」という名称を使用しています。

!!! info "保存期間を10年間とした理由"
    一般社団法人法第129条等の規定により、計算書類（貸借対照表など）は10年間の保存が義務付けられています。法律の要件を満たすため、保存期間を10年間としています。

### 第33条（剰余金の分配の禁止）
この法人は、剰余金の分配を行わない。

### 第34条（特定の個人等への利益供与の禁止）
この法人の事業により利益が生じたときは、これを社員その他の特定の個人又は団体に分配せず、次の事業年度以降の事業活動に充てるものとする。

!!! warning "第33条・第34条は絶対に削除・変更しないこと"
    これらの条項は、当法人が「非営利型法人」として認められるための必須要件です。

    - **第33条（剰余金の分配の禁止）**: 「非営利が徹底された法人」または「共益的活動を目的とする法人」として認められるために、剰余金の分配を行わないことを明記する必要があります。
    - **第34条（特定の個人等への利益供与の禁止）**: 特定の個人に特別の利益を与えないことを誓約するもので、これも非営利型の要件に関わります。

    これらの条項を削除・変更すると、非営利型法人としての税制上の優遇措置を受けられなくなる可能性があります。
