# 会員の設定内容とその理由

### 第6条（会員の種別）
1. この法人の会員は、次の2種とし、正会員をもって一般社団法人及び一般財団法人に関する法律（以下「一般法人法」という。）上の社員とする。
    1. 正会員：この法人の目的に賛同して入会した個人又は団体
    2. 賛助会員：この法人の事業を賛助するために入会した個人又は団体

!!! info "正会員と賛助会員を分けた理由"
    一般社団法人法上の「社員」は、社員総会での議決権を持ち、法人の重要事項（定款変更、解散、役員選任等）を決定する権限があります。

    もし支援者全員を「社員」として登録してしまうと、以下のリスクがあります：

    - **特別決議が成立しなくなるリスク**: 特別決議には「総社員の半数以上かつ議決権の3分の2以上」が必要です。社員の半数以上と連絡がつかなくなると、定款変更も解散もできなくなり、法人運営が行き詰まります。
    - **ガバナンスの複雑化**: 社員数が増えると、総会の招集通知や議決権行使の管理が煩雑になります。

    そのため、議決権を持つ「正会員（＝法律上の社員）」は、連絡が確実に取れ責任ある意思決定に参加できる当事者のみに限定し、一般の支援者は「賛助会員」として管理する設計としています。賛助会員は議決権を持ちませんが、法人の活動を支援いただく重要な存在です。

### 第7条（入会）
1. 正会員になろうとする者は、代表理事が別に定める入会申込書により申し込みを行い、代表理事の承認を得なければならない。
2. 賛助会員になろうとする者は、代表理事が別に定める入会申込書により申し込みを行い、代表理事の承認を得なければならない。

### 第8条（会費）
1. 正会員は、社員総会において別に定める入会金及び会費を納入しなければならない。
2. 賛助会員は、社員総会において別に定める賛助会費を納入しなければならない。

!!! note "会費の金額を定款に書かない理由"
    具体的な金額まで定款に書き込んでしまうと、将来金額を変更するたびに定款変更の手続き（社員総会の特別決議）が必要になり、手間がかかります。

    別途**「会費規程」**を作成し、そこで「設立後しばらくは無料」などのルールを明文化します。

### 第9条（任意退会）
会員は、代表理事に対して退会届を提出することにより、いつでも退会することができる。

### 第10条（除名）
正会員が次の各号のいずれかに該当するときは、社員総会の特別決議によって当該正会員を除名することができる。

1. この定款その他の規則に違反したとき
2. この法人の名誉を傷つけ、又は目的に反する行為をしたとき
3. その他除名すべき正当な事由があるとき

!!! note "除名規定について"
    除名は正会員（＝法律上の社員）に対する規定です。賛助会員については、除名という重い手続きではなく、代表理事の判断により会員資格を取り消すことができるものとして運用します。

### 第11条（会員資格の喪失）
会員が次の各号のいずれかに該当する場合には、その資格を喪失する。

1. 退会したとき
2. 成年被後見人又は被保佐人になったとき
3. 死亡し、若しくは失踪宣告を受け、又は解散したとき
4. ○年以上会費を滞納したとき
5. 除名されたとき（正会員の場合）
6. 総正会員の同意があったとき（正会員の場合）
