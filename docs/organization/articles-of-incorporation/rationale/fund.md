# 基金の設定内容とその理由

### 第30条（基金の拠出等）
1. この法人は、基金を引き受ける者の募集をすることができる。
2. 拠出された基金は、この法人が解散するまで返還しない。
3. 前項の規定にかかわらず、この法人は、次条に定める基金の返還の手続きにより、基金を返還することができる。

### 第31条（基金の返還の手続き）
1. 基金の返還は、定時社員総会の決議に基づき、一般社団法人及び一般財団法人に関する法律第141条第2項に規定する限度額の範囲内で行うものとする。
2. 基金の返還を行う場所及び方法その他の必要な事項については、代表理事が別に定める。

!!! info "「解散まで返さない」けど「返せる」という設計について"
    第2項で「解散するまで返還しない」としつつ、第3項で「返還の手続きにより返還できる」としている理由は、**法人の活動が軌道に乗り、寄付金で十分な剰余金（純資産）が出たタイミングで、設立時に出したお金を自分たちに払い戻す**という出口戦略を作るためです。

    設立時に理事が自己資金で基金を拠出した場合でも、将来的に法人運営が安定した段階で、適切な手続きを経て返還を受けることができます。

!!! info "非営利型の要件を守るポイント"
    一般社団法人の基金は、**「利息を付けて返してはいけない」**というルールがあります。10万円出したら10万円だけ返す。これが「利益の分配」ではないとみなされ、非営利性を保つポイントになります。

    基金の返還に利息や配当を上乗せしてしまうと、実質的に「利益の分配」とみなされ、非営利型法人としての税制上の優遇措置を受けられなくなるリスクがあります。
